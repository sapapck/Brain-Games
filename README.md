# Brain Games

# Description:

Добро пожаловать в мой первый проект! Этот репозиторий состоит из нескольких арифметических консольных игр.

# Setup

**Для запуска игр выполните следующие действия:**

1. Проверьте наличие обновлений в базе данных доступных пакетов: введите в командной строке apt update
2. Установите пакет Node.js: введите в командной строке apt install nodejs
3. Обновите Node.js до последней версии: введите в командной строке apt upgrade nodejs
4. Чтобы иметь возможность устанавливать модули и пакеты вместе с Node.js: введите в командной строке apt install npm
5. Добавьте и проверьте зависимости: введите в командной строке make install
6. Запустите игры: введите в командной строке _'название_игры'_
